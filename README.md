### Descripción del Proyecto

Nexa Servicios S.A.S. presenta dificultades en la gestión de órdenes de trabajo debido al uso de canales informales como WhatsApp, correos electrónicos y hojas de cálculo dispersas.

Este proyecto propone el desarrollo de un Sistema de Gestión Operativa que permita centralizar la información, registrar solicitudes, gestionar tareas, administrar checklist, almacenar evidencias y garantizar la trazabilidad completa de los procesos internos.

### Objetivo General

Desarrollar un sistema de información para la gestión automatizada de órdenes de trabajo que permita centralizar los procesos operativos de Nexa Servicios S.A.S., garantizando control, seguimiento y trazabilidad de la información.
